Customization

Dashboard, overlay, preferences, and runtime paths.

This page groups the user-facing controls that shape how the app behaves after configuration is complete.

Dashboard

  • Open the game grid with Show Dashboard.
  • Use search to filter games quickly.
  • Sort by name, progress, or last update time.
  • Click a game card to load its config.
  • Use the play button for games with configured executables.
  • Background polling can auto-select the active game when its process starts.
  • Esc or the close button restores the main UI and input state.

Customization

  • Choose notification preset and screen position.
  • Select notification sounds and language.
  • Adjust UI scale from 75% to 200%.
  • Adjust achievement duration and sound volume.
  • Toggle hidden descriptions when available.
  • Enable Close to Tray.
  • Configure overlay shortcut and interaction key.
  • Enable or disable overlay controller support from Settings → Advanced → Rendering.
  • Enable or disable achievement screenshots, progress notifications, playtime notifications, and startup behavior.

Runtime data locations

  • %APPDATA%\Achievements\configs
  • %APPDATA%\Achievements\configs\schema
  • %APPDATA%\Achievements\images
  • %APPDATA%\Achievements\ach_cache
  • %APPDATA%\Achievements\ach_cache_meta.json
  • %APPDATA%\Achievements\logs
  • %APPDATA%\Achievements\playtime-totals.json
  • %APPDATA%\Achievements\preferences.json

Keyboard and controller highlights

  • Controller Support includes a Dashboard introduction and controller navigation.
  • Multi-Platform Support V2 + Overlay Controller Support also includes the overlay controller support flow.

Global

Settings: F1 / Ctrl+O. Dashboard: F2 / Ctrl+D. Play: P / Ctrl+Enter.

Overlay

Toggle visibility with the configured overlay shortcut. Toggle interaction mode with the configured interact key.

Overlay controller mode

Xbox: View + Menu toggles overlay, hold LB + RB for control mode. PlayStation uses Share + Options and L1 + R1.

Main screen

Create New Config with Ctrl+N. Toggle options with F3. Navigate with arrow keys or controller D-pad.
